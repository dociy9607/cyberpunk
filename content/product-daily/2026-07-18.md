### Google Cloud Data Agent Kit

- 产品名称：Google Cloud Data Agent Kit
- 值得借鉴的原因：Data Agent Kit 最值得产品经理和创业者关注，因为它没有把“数据 agent”做成又一个独立聊天产品，而是嵌入到数据工程师已经使用的 IDE、CLI 和云资源管理路径里。它的交互策略是把浏览数据、运行查询、构建 pipeline、训练模型等动作变成 agent 可调用的上下文能力，降低了企业用户在控制台、Notebook 和编辑器之间切换的摩擦。商业上，这类扩展能成为云平台留存数据开发者的入口：一旦 agent 工作流绑定 BigQuery、Dataflow、Cloud SQL 等资源，后续的计算、治理和部署消费都会自然回流到云平台。对创业团队的启发是，AI 工具不一定要抢占完整工作台，先成为现有工作流中最关键的“操作层”和“上下文桥”，更容易让专业用户持续使用。
